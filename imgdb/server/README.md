# img-DB server

A web interface for img-DB.

Disclaimer: This server is partially created using AI, but it is not vibe-coded.
At least half of the code is written by hand... the old fashioned way :)
I didn't use any specific LLM or tool, I don't have any favorites.
For every feature I wanted to design, I tried a few different LLMs and I selected the best code.
I always review and edit the AI generated code and I understand everything that's going on inside this project.
